# Bus Management System (BMS)

This project runs on a local server using WampServer.

## Requirements
 WampServer installed
 Web browser (Chrome, Edge, etc.)

## How to Run

1. Install and start WampServer.
2. Copy the project folder into the WampServer **www** directory.
   Example:
   C:\wamp64\www\http-localhost-bms
3. Start WampServer and make sure the icon turns green.
4. Open your browser and go to:

http://localhost/http-localhost-bms

## Description
This website is a Book Management System that runs locally using PHP and MySQL.
